### Overview
Kashi’s HomeFood is my project. In our busy schedule it is difficult to plan for a proper meal, 
its energy and time consuming. My project will display all the HomeFood Catering  in the Saint Louis area.
You can choose your favorite dishes and order for carryout. You can also search for vegan, 
vegetarian, gluten free, allergic free menus. 
In this project the consumer will find the entire catering related search in one umbrella.
Once you  order for carryout it will direct you to checkout. 
And finally once the order is finished we can do door delivery.

### Features
Login 
Users will be able to create accounts and log in to the application. Each user will provide Phone number to receive text messages.

Search
Users will be able to search for Restaurants with customer’s reviews 

Reviews 
User can post there reviews and complaints in the application to improve the services of the application.
